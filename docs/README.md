# 524 Documentation

Centralized reference for product, engineering, and operational documentation supporting the 524 beauty services marketplace.

## Structure

- `setup/` – environment setup, configuration, and deployment guides
- `auth/` – authentication system documentation and setup guides
- `ui/` – user interface components, navigation, and UX patterns
- `architecture/` – system diagrams, integration maps, and API contracts *(planned)*
- `decision-records/` – ADRs capturing significant technical decisions *(planned)*
- `playbooks/` – operational runbooks, incident response, on-call guides *(planned)*
- `product/` – feature specifications, UX flows, customer journey maps *(planned)*

Populate each sub-directory with Markdown documents or linked design assets as the platform matures.
