# 524 Documentation

Centralized reference for product, engineering, and operational documentation supporting the 524 beauty services marketplace.

## Structure

- `architecture/` – system diagrams, integration maps, and API contracts
- `decision-records/` – ADRs capturing significant technical decisions
- `playbooks/` – operational runbooks, incident response, on-call guides
- `product/` – feature specifications, UX flows, customer journey maps

Populate each sub-directory with Markdown documents or linked design assets as the platform matures.
